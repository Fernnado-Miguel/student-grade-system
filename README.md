# 🎓 Student Grade System

A simple **C program** that reads student grades, calculates statistics, and determines if the student passed or failed.

---

## 🚀 Features

* Input of **N grades** chosen by the user
* **Input validation** (grades must be between 0 and 10)
* **Average calculation**
* **Pass / Fail verification**
* **Highest grade detection**
* **Lowest grade detection**
* Count of grades **above the average**
* Count of grades **below the average**

---

## 🧠 How it Works

1. The user chooses how many grades will be entered.
2. The program validates the number of grades.
3. Each grade must be between **0 and 10**.
4. The program calculates:

   * Average grade
   * Highest grade
   * Lowest grade
   * Grades above the average
   * Grades below the average
5. Finally, it displays the results.
